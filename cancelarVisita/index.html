Cancelar Visita#

Se houver algum impedimento para acesso ao tanque resfriador, como o caso de uma porteira trancada, estrada interompida ou outro motivo qualquer, o motorista(agente de coletas) deve registrar o motivo do impedimento, e inclusive registrar uma foto para comprovar a situação.
